# testing positiv
1. Первый вариант позитивного тестирования
  - ввести в поле "имя" данные на кириллице, например (Дмитрий)
  - ввести номер телефона в поле для номера в формате +7 (982)-521-73-93
  - нажать кнопку "Записаться на занятие" с помощью мыши
Ожидание: уведомление об успешной записи на занятие либо ошибка формата телефонного номера.
2. Второй вариант позитивного тестирования
  - ввести в поле "имя" данные на латинице, например (Dmitry)
  - ввести в поле для номера, телефонный номер в формате 89825127393
  - нажать кнопку "Записаться на занятие" с помощью мыши
Ожидание: уведомление об успешной записи на занятие
3. третий вариант позитивного тестирования
  - вставить в поле "имя" данные посредством горячих клавиш ctrl+V, например (Дмитрий)
  - вставить в поле для номера телефона данные посредством горячих клавиш ctrl+V, например (+79825127393)
  - нажать кнопку "Записаться на занятие" с помощью клавиши Enter
Ожидание: уведомление об успешной записи на занятие

# testing negativ
1. Первый вариант негативного тестирования
   - Ввести в поле "имя" данные в виде цифр, например (986348)
   - Ввести в поле для номера телефона корректные данные
   - нажать кнопку "Записаться на занятие" с помощью мыши
   Ожидание: уведомление о некорректном формате введённых данных в поле "имя"
2. Второй вариант негативнго тестирования
   - Ввести в поле "имя" данные на кириллице, например (Дмитрий)
   - Ввести в поле для номера телефона данные в виде букв, например (Дмитрий)
   - нажать кнопку "Записаться на занятие" с помощью мыши
   Ожидание: уведомление о некорректном формате введённых данных в поле для номера телефона
3. Третий вариант негативного тестирования
   - Ввести данные в поле "имя" на кириллице в количестве символов превышающих размер поля
   - Ввести в поле для номера телефона корректные данные
   - нажать кнопку "Записаться на занятие" с помощью мыши
   Ожидание: уведомление о некорректном формате введённых данных в поле "имя"  или предупреждение на этапе ввода слишком большого количества символов.
4. Четвёртый вариант негативного тестирования
   - Ввести в поле "имя" данные на кириллице, например (Дмитрий)
   - Ввести в поле для номера телефона большое количество символов превышающее размер поля
   - нажать кнопку "Записаться на занятие" с помощью мыши
   Ожидание: уведомление о некорректном формате введённых данных в поле для номера телефона или предупреждение на этапе ввода слишком большого количества символов.
5. Пятый вариант негативного тестирования
   - Не вводить данные в поля для заполнения
   - нажать кнопку "Записаться на занятие" с помощью мыши
   Ожидание: уведомление о необходимости ввести данные
